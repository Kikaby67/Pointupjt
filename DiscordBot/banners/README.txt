Dossier des bannières de jeux pour /calendrier.

Nomme chaque image avec le nom du jeu en minuscules, sans espaces ni
ponctuation, extension .png (ou .jpg/.jpeg/.webp).

Exemples (selon ce que tu tapes dans /addstream jeu:...) :
  "Enshrouded"       -> banners/enshrouded.png
  "Terraria"         -> banners/terraria.png
  "Romestead"        -> banners/romestead.png
  "Baldur's Gate"    -> banners/baldursgate.png
  "Just Chatting"    -> banners/justchatting.png

Format conseillé : large (ex 1200x400), l'image est recadrée pour remplir
la case du jour. Si aucun fichier ne correspond, la case affiche le texte
du jeu (ou "REPOS / CONTENU" si le jour est vide).

Quand tu ajoutes un stream, /addstream te rappelle le nom de fichier attendu.
