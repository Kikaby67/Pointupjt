# Stratégie Réseaux Sociaux — Projet Pointu
### *TikTok · Instagram Reels · YouTube Shorts*

---

## Objectif

Créer et fidéliser une communauté de joueurs potentiels avant et pendant les lives Twitch,
en installant **l'univers d'Arbonet** dans l'esprit des gens via du contenu court et accrocheur.

---

## Piliers de Contenu

| Pilier | Fréquence | Objectif |
|--------|-----------|----------|
| 🌍 **Lore Drop** | 1x/semaine | Installer l'univers, créer l'envie |
| 🎮 **Dev Log** | 1x/semaine | Humaniser le projet, montrer la progression |
| 🪝 **Hook Mystère** | 2x/semaine | Curiosité, engagement, partage |
| 📣 **Appel à l'action** | 1x/semaine | Redirection vers Twitch, recrutement joueurs |
| 🤝 **Behind the scenes** | 1x/2 semaines | Connexion personnelle, authenticité |

---

## Tonalité par Plateforme

### TikTok
- Ton : **direct, mystérieux, légèrement dramatique**
- Format : Hook visuel fort dans les 2 premières secondes
- Durée cible : 30–45 secondes
- Hashtags : #twitch #twitchfr #gaming #rpg #indiegame #arbonet #pointu #lore #jeu

### Instagram Reels
- Ton : **esthétique, story-telling soigné**
- Format : Visuellement cohérent (couleurs Arbonet : vert électrique, noir, argent)
- Durée cible : 30–60 secondes
- Hashtags : #twitch #gaming #rpg #worldbuilding #indiedev #lore #jeu #communauté

### YouTube Shorts
- Ton : **plus posé, un peu plus explicatif**
- Format : Peut aller jusqu'à 60 secondes avec plus de contexte
- Durée cible : 45–60 secondes
- Tags : RPG Twitch, mini-jeu Twitch, lore gaming, game dev solo

---

## Charte Visuelle Recommandée

- **Palette couleurs** : Vert électrique (#00FF88), Noir profond (#0A0A0A), Argent (#C0C0C0)
- **Police** : Monospace ou pixel pour les textes de lore / classique pour la narration
- **B-roll idéal** : Forêt la nuit, racines lumineuses, code sur écran, turtle pixel art
- **Musique** : Ambient electronique + nature, mysterious lo-fi, cyberpunk forest vibes

---

## Funnel Communauté

```
Short/Reel/TikTok
       ↓
   Curiosité → Commentaire → Follow
       ↓
   Bio / Lien → Twitch
       ↓
   Live → !rejoindre → Joueur enregistré
       ↓
   Communauté active → Partage organique
```

---

## KPIs à Suivre

- Taux de completion des vidéos (> 70% = bon)
- Commentaires avec questions sur le lore (signe d'engagement réel)
- Clics vers Twitch depuis la bio
- Nombre de joueurs enregistrés via `!rejoindre` après un post
- Partages spontanés
