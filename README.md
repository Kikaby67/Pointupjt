# Pointu-pjt
Ce projet est fait et réalser en solo dans le but d'un apprentiswsage de la progrommatation.
Je me sert de ce projet pour créer un Mini jeu afin de créer de l'inéraction avec les personnes aui suivent un live sur twitch.
J'ai créer tout un lore qui sera la référence pour ce mini jeu.
J'utilise la plateforme de srteaming Twitch, Obs pour enregistrer et diffuser le live et Streamer.bot pour gérer les alertes, commandes et autres joyeusetés.
Le language de programmation est c#.
Openclassrooms me sert de support d'apprentissage avec leurs cours en free-access. Je m 'aide également de Claude.IA pour ajouter un outil d'apprentissage et de mise en place.
