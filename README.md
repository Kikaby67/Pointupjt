# Pointu-pjt
Ce projet est fait et réaliser en solo dans le but d'un apprentissage de la programmation.
Je me sert de ce projet pour créer un Mini jeu afin de créer de l'interaction avec les personnes qui suivent un live sur Twitch. J'ai créer tout un Lore qui sera la référence pour ce mini jeu.
J'utilise la plateforme de streaming Twitch, Obs pour enregistrer et diffuser le live et Streamer.bot pour gérer les alertes, commandes et autres joyeusetés.
Le langage de programmation est c#. Openclassrooms me sert de support d'apprentissage avec leurs cours en free-access.
Je m 'aide également de Claude. IA pour ajouter un outil d'apprentissage et de mise en place.
